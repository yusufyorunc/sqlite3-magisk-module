# 适用于 arm、arm64、x86 和 x64 的 SQLite3

## 描述

该 Magisk 模块提供了一个静态链接并优化的 ´sqlite3´ 二进制文件，适用于 **arm64-v8a**、**armeabi-v7a**、**x86** 和 **x86_64** 设备。

## 测试设备

 - arm64-v8a:
   - **Lenovo Tab M10 Plus Gen 3**
   - **Xiaomi MI 6**
   - **Samsung Galaxy S6 Edge**
   - **Poco X3 Pro**
 - x86_64:
   - **LDPlayer 9**
   - **Waydroid**
