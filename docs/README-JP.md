# arm、arm64、x86、および x64 用の SQLite3

## 説明

この Magisk モジュールは、**arm64-v8a**、**armeabi-v7a**、**x86**、および **x86_64** デバイス向けに静的にリンクされ、最適化された ´sqlite3´ バイナリを提供します。

## テスト済みデバイス

 - arm64-v8a:
   - **Lenovo Tab M10 Plus Gen 3**
   - **Xiaomi MI 6**
   - **Samsung Galaxy S6 Edge**
   - **Poco X3 Pro**
 - x86_64:
   - **LDPlayer 9**
   - **Waydroid**
