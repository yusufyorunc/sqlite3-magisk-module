# arm, arm64, x86 및 x64용 SQLite3

## 설명

이 Magisk 모듈은 **arm64-v8a**, **armeabi-v7a**, **x86** 및 **x86_64** 장치를 위한 정적으로 링크되고 최적화된 ´sqlite3´ 바이너리를 제공합니다.

## 테스트된 기기

 - arm64-v8a:
   - **Lenovo Tab M10 Plus Gen 3**
   - **Xiaomi MI 6**
   - **Samsung Galaxy S6 Edge**
   - **Poco X3 Pro**
 - x86_64:
   - **LDPlayer 9**
   - **Waydroid**
